Linked Lists in C

Linked lists in C are a great way to store data sets that require frequent inserts, deletions, and changes in length. You might consider a linked list over an array because inserting or deleting elements would involve moving every element in the array after the item of concern.

The code sample here demonstrates a linked list in C implemented with structs. It includes some of the standard list operations such as: inserting at a given position, deleting at a given position, and deleting an item by value
